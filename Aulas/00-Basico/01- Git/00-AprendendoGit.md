# Introdução ao GitHub

---

## Versionamento de Código 
- A cada mudança no código, é criada uma **nova versão**.  
- O versionamento ajuda a **controlar o histórico**, voltar a versões anteriores e colaborar com outras pessoas sem perder nada.

---

## Sistema de Controle de Versão
- Registra o **histórico de atualizações**.  
- Gerencia **o que foi alterado**.  
- Organiza, controla e deixa o código **mais seguro**.
